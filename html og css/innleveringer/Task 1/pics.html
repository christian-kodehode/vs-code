Site info

The album gallery contains a number of images of various lengths and heights. Shown in the gallery are thumbnails of each image cropped to size 250x250 px and center-aligned horizontally and vertically.

Each thumbnail has a 2 pixel black border and is also a hyperlink to its corresponding fullsized image. When the mouse hovers over the thumbnail its opacity is changed to 60% to easily show which thumbnail is "marked".

Each thumbnail also has an image description attached below. A thumbnail and its corresponding image description are placed within its own div-element.
